本《衡法智核 LexCore 隐私政策》（以下简称“本政策”）旨在向您说明，在您使用“衡法智核 LexCore”服务过程中，我们如何收集、使用、存储、共享、转让、公开披露和保护您的个人信息，以及您如何管理您的个人信息和行使相关权利。

## 一、我们如何收集和使用您的个人信息

我们会基于实现产品基本功能、履行法定义务、保障交易与账号安全、优化服务体验等目的，在必要范围内收集和使用您的个人信息。

### 1. 账号注册与登录

当您注册、登录或绑定账号时，我们可能收集以下信息：
- 手机号码；
- 邮箱地址；
- 登录密码或其加密后的校验信息；
- 验证码信息；
- 第三方账号授权信息（如您选择第三方登录）。

**使用目的：**
- 帮助您完成账号注册、登录、身份识别与安全验证；
- 用于账号找回、风险核验、客服处理和消息通知。

如您拒绝提供上述必要信息，您可能无法注册或登录，但不影响您浏览部分不依赖账号的基础功能。

### 2. 法律咨询、文书生成、案件分析等核心功能

当您使用法律问答、法条检索、案件分析、文书生成、历史记录、导出下载等功能时，我们可能收集您主动输入、上传或形成的内容，包括：
- 咨询问题、检索关键词；
- 案件事实描述、争议焦点说明；
- 文书需求、合同条款、当事人信息；
- 您上传的文档、图片、附件及其文本内容；
- 您对生成结果的修改、保存、导出、收藏、删除操作记录。

**使用目的：**
- 实现法律问答、检索、分析、生成、预览、导出等功能；
- 向您展示处理结果并保存历史记录；
- 提升回复质量、问题定位和服务稳定性；
- 在必要时进行合规审查、风险控制和安全防护。

请您注意：如您在输入内容中主动提交身份证号、联系方式、住址、银行卡信息、案件材料、未成年人信息、健康信息、诉讼材料等，相关内容可能属于敏感个人信息。请您仅在确有必要且具备合法依据时提交。我们建议您避免输入与当前服务无关的敏感信息。

### 3. 客服与反馈

当您与我们联系、提交意见反馈、投诉举报或申请售后支持时，我们可能收集：
- 您主动提供的联系方式；
- 反馈内容、问题描述、截图、附件；
- 处理记录、沟通记录。

**使用目的：**
- 帮助您排查和解决问题；
- 响应您的权利请求；
- 留存服务记录以备查询和争议处理。

### 4. 设备信息、日志信息与安全风控

为保障服务安全、稳定运行并防范欺诈、攻击、异常调用等风险，我们可能收集：
- 设备型号、操作系统版本、应用版本号；
- 设备标识信息（如 Android ID、OAID，具体以实际接入情况为准）；
- 网络类型、IP 地址、接入时间、崩溃日志、操作日志、服务日志；
- 页面访问记录、功能使用记录、请求链路标识、异常诊断信息。

**使用目的：**
- 识别账号安全风险和异常环境；
- 进行故障排查、性能优化和接口稳定性分析；
- 防止恶意注册、滥用接口、刷量、攻击、作弊等行为。

### 5. 通知与消息推送

如您开启消息通知权限，我们可能向您发送：
- 登录提醒、安全通知；
- 结果处理提醒、导出完成提醒；
- 服务变更、协议更新、活动通知等。

您可以在设备系统设置或应用内设置中关闭相关通知。

## 二、我们可能申请的设备权限

为了实现安卓 App 的特定功能，我们可能在您主动使用相应功能时申请以下权限。不同系统版本下权限展示方式可能有所不同，最终以您设备实际弹窗为准。

### 1. 存储/文件读取与写入权限
**使用场景：**
- 上传本地文件、附件、图片；
- 保存生成的文书、导出文件；
- 缓存必要的应用数据。

**拒绝影响：**
如您拒绝授权，您可能无法上传本地文件或保存导出结果，但不影响其他基础功能使用。

### 2. 相机权限
**使用场景：**
- 拍摄图片、扫描纸质文件；
- 采集您主动拍摄后上传的材料。

**拒绝影响：**
如您拒绝授权，您将无法使用拍照上传、扫描相关功能，但仍可通过其他方式上传文件。

### 3. 相册/媒体访问权限
**使用场景：**
- 从系统相册选择图片、截图或文档用于上传。

**拒绝影响：**
如您拒绝授权，您将无法从本地相册选择相应内容。

### 4. 剪贴板读取
**使用场景：**
- 仅在您主动触发粘贴操作时，读取您复制的内容以便填写表单或导入文本。

**拒绝影响：**
不会影响主要功能，但您可能需要手动输入相关内容。

### 5. 通知权限
**使用场景：**
- 发送账号安全提醒、结果通知、系统消息。

**拒绝影响：**
您仍可正常使用服务，但可能无法及时收到通知。

> 我们不会因为您不同意非必要权限，而拒绝您使用与该权限无关的功能。

## 四、我们如何共享、转让、公开披露您的个人信息

### 1. 共享

我们不会向第三方出售您的个人信息。我们仅会在以下情形下共享您的个人信息：

1. 在获得您的单独同意或授权的情况下；
2. 为实现核心服务功能，与受托服务商、云服务商、短信服务商、消息推送服务商、第三方 SDK 服务商共享必要信息；
3. 为履行法定义务、响应司法或行政机关要求，依法提供；
4. 为保护您、其他用户、社会公众或我们的合法权益所必要；
5. 在合并、分立、重组、收购、资产转让等交易中，如涉及个人信息转移，我们会要求接收方继续受本政策约束。

### 2. 转让

原则上，我们不会将您的个人信息转让给任何公司、组织和个人；确需转让的，我们会依法履行告知义务并取得必要同意。

### 3. 公开披露

我们原则上不会公开披露您的个人信息。确需公开披露时，我们会依法履行相应程序，并采取符合行业标准的安全保护措施。

## 六、我们如何存储您的个人信息

1. 我们会在实现本政策所述目的所必需的期限内保存您的个人信息，法律法规另有规定的除外。
2. 超出保存期限后，我们将对您的个人信息进行删除或匿名化处理，除非法律法规另有要求。
3. 我们将采取合理可行的措施，尽量避免收集与服务无关的个人信息。
4. 您的个人信息原则上存储在中华人民共和国境内服务器。若未来确需进行跨境传输，我们将依法履行相应义务并向您告知。

## 七、我们如何保护您的个人信息

1. 我们会采取加密传输、访问控制、权限分级、日志审计、备份恢复、安全监测等合理安全措施保护您的个人信息。
2. 我们会建立内部管理制度，对接触个人信息的工作人员进行权限控制和安全要求管理。
3. 尽管如此，互联网环境并非绝对安全，我们无法保证任何信息在传输或存储过程中百分之百安全。请您妥善保管账号、密码和验证码。
4. 如发生或可能发生个人信息安全事件，我们将依法采取补救措施，并按照法律法规要求向您告知相关情况。

## 八、您的个人信息权利

按照适用法律法规，您对您的个人信息享有以下权利：

### 1. 查询和访问
您有权查询、访问您的注册信息、历史记录、文书记录、案件分析记录等。

### 2. 更正和补充
当您发现您的个人信息不准确或不完整时，您有权要求更正或补充。

### 3. 删除
在法律法规规定的情形下，您有权请求删除您的个人信息，例如：
- 处理目的已实现、无法实现或不再必要；
- 您注销账号；
- 我们违反法律法规或约定处理您的个人信息；
- 其他法律法规规定的情形。

### 4. 复制和导出
在符合法律法规和技术可行的前提下，您有权获取您个人信息的副本。

### 5. 撤回同意
对于基于您同意处理的个人信息，您有权撤回相应同意。撤回后，不影响撤回前基于您同意已进行的处理活动的效力。

### 6. 注销账号
您可以通过应用内路径或联系我们申请注销账号。  
建议在 App 内提供明确路径，例如：  
**我的 → 设置 → 账号与安全 → 注销账号**

账号注销后，我们将停止为您提供服务，并按法律法规要求删除或匿名化处理您的个人信息，但法律法规另有规定的除外。

### 7. 响应您的请求
如您需要行使上述权利，可通过本政策载明的联系方式与我们联系。为保障安全，我们可能要求您进行身份核验。我们将在法律法规规定期限内处理并回复您的请求。

4. 如监护人对未成年人个人信息保护有疑问、意见或请求，可通过本政策中的联系方式与我们联系。
