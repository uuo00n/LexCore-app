# 衡法智核 LexCore 项目 IP 方案

## 一、IP总定位

**中文名称：衡法智核**
**英文名称：LexCore**
**项目副标题：智能法律服务平台**

### 1. 品牌定位

衡法智核 LexCore 是一个面向法律服务场景的智能化平台型产品，围绕法律咨询、法律文书生成、案件分析、法条检索等核心需求，提供专业、精准、可扩展的智能法律服务能力。项目强调“多智能体协同 + 智能中枢调度 + 标准化输出”，具备从比赛展示到后续产品化落地的双重属性。

### 2. 品牌关键词

**专业、可信、理性、智能、协同、中枢、稳健、科技**

### 3. 核心价值主张

**以智能内核重塑法律服务效率，让法律服务更专业、更精准、更可及。**

------

## 二、命名释义体系

## 1. 中文名释义：衡法智核

### 衡

代表衡平、公正、权衡、审慎，体现法律服务对于事实、规则与责任的理性判断。

### 法

代表法律专业属性，直接锚定项目所属领域，建立清晰的行业识别度。

### 智核

代表智能内核、能力中枢、系统核心，契合你项目的 AI 编排层、多智能体调度机制和平台型架构。项目方案中明确存在 supervisor、专业 agent、formatter 统一输出等中枢式设计，因此“智核”不是空泛修饰，而是对应真实系统结构。

### 中文名整体释义

**衡法智核** 即“以智能核心能力支撑法律判断与服务协同的专业平台”。

------

## 2. 英文名释义：LexCore

### Lex

源自拉丁语中的法律、法条语义，国际化表达简洁直接。

### Core

表示核心、内核、中枢，与“智核”语义高度统一。

### 英文名整体释义

**LexCore** 表示“法律智能核心”或“法律服务的智能中枢”。

## 四、品牌故事

在法律服务领域，长期存在专业门槛高、服务成本高、资源分布不均、处理效率有限等问题。与此同时，法律咨询、法律文书生成、案件分析和法规检索等工作又高度依赖严谨的逻辑能力与结构化表达能力。衡法智核 LexCore 由此诞生：它不是一个普通聊天机器人，而是一个围绕法律服务流程构建的智能中枢平台。系统通过多智能体协同架构，将法律问答、文书生成、案件分析、法条检索等能力进行专业化拆分，再由统一调度与格式化模块实现高质量输出，从而推动法律服务从“单点辅助”走向“系统化协同”。这一定位与你现有方案中的多场景法律服务目标、多智能体路由设计和标准化 JSON 输出结构是吻合的。

## 五、品牌世界观与人格设定

## 1. 品牌世界观

法律服务不应只是少数专业人群可高成本获取的能力，而应借助可信的智能系统，成为更加高效、规范、可触达的公共服务能力。

## 2. 品牌人格

**理性、克制、专业、可信、沉稳、现代**

## 3. 品牌语气

对外表达应避免浮夸、娱乐化、强营销化措辞。
推荐使用：

- 专业表述
- 简洁判断
- 结构清晰
- 强调依据与效率
- 强调辅助而非替代

### 语气示例

正确方向：
“基于法律检索与智能分析能力，提供结构化法律服务支持。”

不建议：
“最强法律 AI”“颠覆行业”“秒杀律师”
这类表达会破坏品牌可信度，也不符合智慧法律场景。

## 六、视觉识别方案

## 1. 品牌气质

视觉上应体现：
**法治秩序感 + 科技理性 + 中枢架构感**

## 2. 推荐主色

### 主色

**深靛蓝 / 法律蓝**
用于主品牌色，体现理性、信任、专业。

### 辅色

**银灰 / 石墨灰**
用于高级感和秩序感。

### 点缀色

**冷金 / 青蓝**
少量用于强调按钮、高光、图表重点区域。

## 3. 推荐视觉元素

- 天平抽象线条
- 中枢节点网络
- 法条页签 / 文书版式框架
- 几何网格
- 微发光数据流
- 核心中轴对称布局

## 4. Logo方向建议

### 方案A：字母型

以 **L + C** 组合形成中轴或盾牌轮廓，表达 LexCore 的平台中枢感。

### 方案B：图形型

使用“天平 + 芯片 + 中心节点”融合图形，体现“法律 + 智能内核”。

### 方案C：中文印记型

以“衡”或“核”的结构做几何化重构，偏东方高级感，适合答辩封面和海报。

## 5. 字体风格建议

中文用偏现代、稳重、横平竖直的无衬线字体。
英文标题可用简洁现代的无衬线体。
避免花体、圆体、卡通体。
